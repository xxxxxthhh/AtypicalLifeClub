## 执行摘要

Synopsys 是 P2 的 EDA/IP 公共约束报告。如果 AI silicon 需求真实，它不只会出现在 foundry capex 和半导体设备订单里，也会出现在 design automation、verification、simulation、IP blocks 和 system-level complexity 里。SNPS 验证的是设计启动与复杂度这一侧。

FY2026 Q2 收入 $2.276B，高于指引；去年同期为 $1.604B。GAAP EPS $0.09，non-GAAP EPS $3.35。公司指引 FY2026 revenue $9.625-9.705B，non-GAAP EPS $14.72-14.80，operating cash flow 约 $2.3B，free cash flow 约 $2.0B，capex 约 $300M。FY2026 revenue 指引包含约 $2.96B 预期 Ansys revenue，其中包括约 $60M channel / accounting impact。

按 2026-07-02 收盘 $437.16，SNPS 市值约 $83.7B，EV 约 $92.1B。按 FY2026 revenue 中点 $9.665B，EV/sales 约 9.5x；按 non-GAAP EPS 中点 $14.76，P/E 约 29.6x。

## 1. 业务与链条角色

SNPS 把 AI compute architecture 连接到设计它所需的软件基础设施。Design Automation 包括 silicon design、verification、system integration、digital/custom/FPGA IC design software、manufacturing software 和 Ansys products；Design IP 包括 logic libraries、embedded memories、wired and memory interface IP、security IP 和 embedded processors。

因此 Synopsys 应该和半导体设备一起放在 P2。工具订单说明晶圆厂是否在建产能；EDA 和 IP 说明客户是否真的在启动、验证、仿真并 tape out 越来越复杂的芯片和系统。

## 2. 牛熊框架

**牛市逻辑：** AI scaling 提高芯片架构多样性和系统复杂度，扩大 verification、simulation、IP 和 multiphysics 需求。Ansys 收购把 SNPS 从 chip design 扩展到 system simulation，提高 AI infrastructure complexity 形成更大软件约束的概率。

**熊市逻辑：** Q2 GAAP EPS 只有 $0.09，说明收购和整合影响很重。Ansys integration 很大，收入指引包含 acquired revenue，公司还明确假设没有进一步 export-control 或 Entity List 变化。任何整合拖累或中国/出口管制收紧，都会削弱设计启动读数。

## 3. 核心监控

1. **Ansys 整合：** FY2026 指引包含约 $2.96B Ansys revenue。关键是整合扩大平台，还是稀释执行。
2. **Design Automation 增长：** 核心信号是 AI design complexity 是否推动 verification、manufacturing software 和 simulation 需求。
3. **Design IP 拉动：** memory interface IP、embedded memories、security IP、processors 应在 custom AI silicon 扩散时受益。
4. **出口管制假设：** 指引假设没有进一步 export-control / Entity List 变化，这是活风险。
5. **FCF 转化：** FY2026 free cash flow 指引约 $2.0B；整合噪音正常化后，现金转化需要恢复。

## 4. 估值背景

价格台账输入：2026-07-02 收盘 $437.16，市值约 $83.7B，EV 约 $92.1B。FY2026 revenue 中点 $9.665B，non-GAAP EPS 中点 $14.76，对应约 9.5x EV/sales 和 29.6x non-GAAP P/E。

SNPS 相对 CDNS 的 forward sales 和 earnings 倍数更低，因为 Ansys integration 和 GAAP 噪音更大。只有 Ansys 能扩大 design / simulation stack，且出口管制没有破坏指引，这个估值才合理。

## 5. 结论

SNPS 是 EDA/IP 公共约束，是 ASML 制造瓶颈的设计侧对应物。评级为 **neutral-watch**：AI 复杂度支撑长期逻辑，但 Ansys integration、出口风险和 acquired-revenue 质量都需要严密监控。
