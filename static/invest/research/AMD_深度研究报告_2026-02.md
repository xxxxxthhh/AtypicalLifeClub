## 1. 业务概览

### 1.1 AMD 做什么

AMD 是高性能计算与图形处理芯片公司，核心业务分为四块：

**数据中心（2025Q4收入占比约52%）：**
- EPYC 服务器CPU（x86，与Intel竞争）
- Instinct AI加速卡（MI300/MI350/MI400，与NVIDIA竞争）
- 2025Q4收入：54亿美元（+39%）

**客户端（2025Q4收入占比约30%）：**
- Ryzen 桌面/移动 CPU
- 2025Q4收入：31亿美元（+34%）

**游戏（2025Q4收入占比约8%）：**
- Radeon 独立显卡
- PlayStation / Xbox 半定制芯片
- 2025Q4收入：8.43亿美元（+50%）

**嵌入式（2025Q4收入占比约9%）：**
- Xilinx FPGA / 自适应 SoC
- 工业、车载、航天等嵌入式处理器
- 2025Q4收入：9.50亿美元（+3%）

### 1.2 商业模式

AMD 采用 **Fabless（无晶圆厂）** 模式：主要负责设计，制造依赖台积电等代工厂。

- **优势：** 资本开支轻、制程迭代快、运营更灵活
- **风险：** 制造集中度高（台湾）、供应链依赖强、制造可控性弱

收入来源以产品销售为主，客户包括 OEM、云厂商、系统集成商与分销渠道。不同赛道的定价权不一致：
- AI 数据中心：相对更强
- 客户端 CPU：竞争更激烈

### 1.3 2025年表现

- 营收：346亿美元（+34%）
- 毛利率（Non-GAAP）：54-57%
- 营业利润（Non-GAAP）：78亿美元
- 净利润（Non-GAAP）：68亿美元（+42%）
- EPS（Non-GAAP）：5.53（+40%）
- 自由现金流：56亿美元（较2024年几乎翻倍）

## 2. 行业与竞争位置

### 2.1 TAM（可服务市场）

**AI加速器（数据中心GPU）市场：**
- 2026规模：约270-370亿美元
- 2034规模：约1250-3250亿美元
- CAGR：约22-36%

**服务器CPU市场：**
- 2026规模：约139亿美元
- 增速：中个位数
- 结构变化：x86主导地位受到Arm渗透挑战

**更大维度的AI基础设施：**
- 2026年超大云厂商 AI 资本开支：约4500亿美元

### 2.2 市场份额

**AI加速器：**
- NVIDIA：约86-90%
- AMD：约5-8%（从低基数快速增长）
- Intel：约2-3%

**服务器CPU：**
- Intel：仍是第一但份额下滑
- AMD：自2018年以来持续拿份额，估计20-25%
- Arm：为潜在中长期增量威胁

### 2.3 护城河评估：中等（Moderate）

**优势：**
1. 芯粒化架构与先进制程协同
2. x86 双寡头格局中的一极
3. 规模带来的研发杠杆提升
4. 与微软、Meta、Oracle、OpenAI 等客户绑定加深

**劣势：**
1. ROCm 生态成熟度显著落后 CUDA
2. 在AI市场仍被视为“替代方案”
3. 制造高度依赖台积电
4. 为争单常需较 NVIDIA 有折价

**结论：** CPU 护城河在增强，AI GPU 护城河仍偏脆弱，关键取决于软件生态补齐速度。

## 5. 牛市论点（Steel-Manned Bull Case）

### 核心论点

AMD 正在 AI 加速器与服务器 CPU 双线推进，若 AI 基础设施开支维持高位，公司有望进入多年的高增长与利润扩张阶段。

### 关键证据

1. **AI业务拐点出现**
- MI300/MI350部署提速，客户覆盖扩张
- OpenAI 多年协议被市场视为中长期增量锚

2. **CPU业务是现金牛**
- EPYC 持续拿份额，为 AI 投入提供稳定现金流

3. **利润率有继续上行空间**
- 产品结构优化 + 规模效应强化

4. **财务弹性足够**
- 低杠杆、净现金、较高FCF

5. **管理层兑现记录强**
- 历史上跨周期执行力已被验证

### 牛市成立的前提

1. AI开支在2026-2027不显著下修
2. OpenAI部署在2026下半年顺利放量
3. ROCm 生态持续改善
4. 台积电产能与先进制程供给稳定
5. 未出现重大地缘冲击

### 上行情景（2027）

- 营收：550-650亿美元
- 营业利润率：32-35%
- EPS：8-10
- 估值（25x forward）：股价潜在200-250美元区间

## 6. 熊市论点（Steel-Manned Bear Case）

### 核心论点

AMD 当前估值已计入较多乐观假设，但 NVIDIA 的软件生态优势、AI 开支周期波动、供应链地缘风险和执行门槛都可能导致不对称下行。

### 关键证据

1. **CUDA 护城河深**
- 开发者生态规模差距巨大
- AMD 常以价格换份额，盈利弹性受限

2. **AI投资存在周期回落风险**
- 若 ROI 验证不及预期，CapEx 可能放缓

3. **2026关键节点多、容错低**
- OpenAI放量、MI400节奏、客户集中度均是敏感点

4. **地缘与供应链风险是“生存级”风险**
- 制造集中在台湾，外部冲击可能造成大级别估值重定价

5. **估值不给犯错空间**
- 高估值下，任何节奏偏差都可能带来明显回撤

### 下行情景（2027）

- 营收：380-420亿美元
- 营业利润率：22-25%
- EPS：3.5-4.0
- 估值（22x forward）：股价潜在75-90美元

## 10. 结论与建议

### 综合结论

AMD 处于战略升级关键期，基本面改善明确，但估值已较多反映乐观预期。最合理的策略不是“盲目追高”，而是围绕关键里程碑动态评估。

### 投资建议

- **已有仓位：** 可继续持有，但需严格跟踪2026Q2-Q3验证点
- **新建仓位：** 更适合等待回调或关键催化确认后分批布局
- **仓位控制：** 建议控制在组合3-5%以内

### 未来12个月重点监控

1. 数据中心收入与毛利率趋势
2. OpenAI实际部署节奏
3. ROCm生态与客户案例
4. 竞争格局与产品代际差
5. 地缘风险变化
